# تشغيل المشروع

## المتطلبات
- Node.js
- MongoDB (شغّله محلياً أو استخدم MongoDB Atlas)

## خطوات التشغيل

### 1. تشغيل Backend
```
cd backend
npm start
```
الـ Backend يشتغل على: http://localhost:5000

### 2. تشغيل Frontend (في terminal جديد)
```
cd frontend
npm run dev
```
الموقع يشتغل على: http://localhost:5173
